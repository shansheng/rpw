frontend3 配色与布局优化建议

基于你提供的 8 张中国交建（CCCC）风格低代码平台截图，对 frontend3（Vben Admin）当前风格进行对比分析，并提出可落地的配色与布局优化方案。

核心结论：frontend3 默认使用深色模式 + Ant Design 亮蓝主色（#0072E5），整体风格偏现代、活泼；参考界面采用浅色模式 + 深蓝主色（#005BAC/#003A8C），更稳重、企业级。建议切换到浅色主题，并将主色、侧边栏、卡片、表单、表格等统一向中国交建风格靠拢。

二、frontend3 当前问题分析

  • 默认深色模式：packages/@core/preferences/src/config.ts 中 theme.mode: 'dark'，与参考的浅色企业风格不符。
  • 主色偏亮：默认 colorPrimary: hsl(212 100% 45%)（约 #0072E5），视觉偏年轻、互联网化，不够沉稳。
  • 文字对比度：深色卡片 #1C1C21 + 次要文字 #9CA3AF，长时间阅读对比度偏低。
  • 圆角过大：默认 radius: 0.5（约 8px），卡片甚至 12px+，与参考的方正风格有差距。
  • 表单间距：字段间距较松散，标签常左对齐，参考界面更紧凑且多为顶部/右对齐。
  • 侧边栏：当前深色/纯黑侧边栏，图标文字层级弱；参考为深蓝+白字，选中高亮清晰。
  • 移动端响应：卡片式布局在小屏下容易拥挤，建议改为单列通栏。

三、设计对比图

1. 整体布局与主题模式

建议：将 theme.mode 从 dark 改为 light；主色从 #0072E5 改为 #005BAC；侧边栏从深黑改为 #003A8C。

2. 配色方案

3. 表格与列表

建议：表格背景改为白色，增加斑马纹；状态标签启用 #E6F4FF 底 + #005BAC 字，禁用 #F5F5F5 底 + #6B7280 字；操作列使用蓝色文字链接而非实心按钮。

4. 表单布局

建议：表单卡片改为白色背景、小圆角；标签改为顶部对齐，必填项加红色星号；字段间距从 24px 压缩到 16px；主按钮使用 #005BAC。

5. 卡片、间距与响应式

建议：圆角从 8~12px 降至 2~4px；卡片间距从 16~24px 压缩到 12px；移动端卡片改为单列通栏，触控区域不小于 44px。

6. 侧边栏与导航

建议：侧边栏背景改为 #003A8C，文字图标白色；选中项使用 #005BAC 高亮背景；一级菜单与二级菜单通过缩进和透明度区分。

五、实施路径

建议分两步走：先通过 Vben 偏好配置切换全局主题与主色，再针对具体页面（表格、表单、卡片）做局部样式覆盖。

Step 1：修改全局偏好

编辑 frontend3/apps/web-antd/src/preferences.ts，添加主题覆盖：

export const overridesPreferences = defineOverridesPreferences({
  app: {
    accessMode: 'backend',
    name: import.meta.env.VITE_APP_TITLE,
    enableRefreshToken: true,
  },
  theme: {
    mode: 'light',
    colorPrimary: '#005BAC',
    colorSuccess: '#52C41A',
    colorWarning: '#FAAD14',
    colorDestructive: '#F5222D',
    radius: '0.25',
    semiDarkSidebar: false,
  },
  sidebar: {
    width: 220,
    collapsed: false,
  },
  logo: {
    source: '/cccc-logo.svg',
    sourceDark: '/cccc-logo.svg',
  },
  footer: {
    enable: false,
    fixed: false,
  },
  copyright: {
    companyName: import.meta.env.VITE_APP_TITLE,
    companySiteLink: 'https://gitee.com/yudaocode/yudao-ui-admin-vben',
  },
});

Step 2：修改默认配置（可选，影响所有 app）

如需让其他 app（web-antdv-next、web-ele）也默认使用新风格，修改 frontend3/packages/@core/preferences/src/config.ts：

theme: {
  builtinType: 'default',
  colorDestructive: 'hsl(0 84% 60%)',
  colorPrimary: 'hsl(208 100% 37%)', // #005BAC
  colorSuccess: 'hsl(100 60% 45%)',
  colorWarning: 'hsl(38 92% 50%)',
  mode: 'light',
  radius: '0.25',
  fontSize: 14,
  semiDarkHeader: false,
  semiDarkSidebar: false,
  semiDarkSidebarSub: false,
},

Step 3：侧边栏深色蓝覆盖

在 frontend3/apps/web-antd/src/theme.css（新建）或 internal/tailwind-config/src/theme.css 中添加：

/* 中国交建深蓝侧边栏 */
--color-sidebar: 216 100% 27%;   /* #003A8C */
--color-sidebar-deep: 216 100% 27%;
--color-primary: 208 100% 37%;   /* #005BAC */
--background: 220 14% 96%;       /* #F0F2F5 */
--card: 0 0% 100%;               /* #FFFFFF */
--border: 0 0% 85%;              /* #D9D9D9 */
--foreground: 220 13% 18%;       /* #1F2937 */
--muted-foreground: 220 9% 46%;  /* #6B7280 */

Step 4：局部组件微调

  • 表格：全局覆盖 .vxe-table 或 .ant-table 行背景、表头背景、斑马纹。
  • 表单：统一表单项下边距为 16px，标签顶部对齐。
  • 卡片：减小圆角到 4px，间距 12px。
  • 状态标签：使用 Tag 组件的 color="blue" 或自定义 #E6F4FF 底。

Step 5：验证与清理

  • 清空浏览器缓存（Vben 偏好持久化在 localStorage，修改后需清理缓存）。
  • 检查深色模式下 logo 与侧边栏文字对比度。
  • 在 1366px、1920px 及移动端分辨率下验证布局。

六、风险与注意事项

  • 缓存问题：Vben 的偏好设置会持久化到 localStorage，修改配置后必须清空浏览器缓存或调用偏好重置。
  • 组件库差异：web-antd 使用 Ant Design Vue，web-antdv-next 使用 Ant Design Vue 4.x，web-ele 使用 Element Plus。同一套色值需要分别在各组件库中映射。
  • 暗色模式兼容：若需保留暗色模式，建议为中国交建蓝准备一套暗色 token（如 sidebar 用 #001529，主色 #1677FF）。
  • 表单字段调整：从标签左对齐改为顶部对齐可能影响表单宽度，需要同步调整 labelWidth 和栅格。
